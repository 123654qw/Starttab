设置帮助

请将您的浏览器主页设置为：https://starttab.rth1.xyz/

Chrome / Edge 浏览器

  1. 点击浏览器右上角的三个点图标
  2. 选择「设置」
  3. 在左侧菜单中选择「外观」
  4. 开启「显示主页按钮」
  5. 点击「更改」，输入 https://starttab.rth1.xyz/
  6. 在「启动时」选项中选择「打开特定页面或一组页面」
  7. 点击「添加新页面」，输入 https://starttab.rth1.xyz/

Firefox 浏览器

  1. 点击浏览器右上角的三个横线图标
  2. 选择「设置」
  3. 在「常规」选项卡中找到「启动时」
  4. 选择「显示主页」
  5. 在「主页」栏中输入 https://starttab.rth1.xyz/

360浏览器

  1. 点击浏览器右上角的工具图标
  2. 选择「选项」
  3. 在「基本设置」中找到「启动时打开」
  4. 选择「主页」
  5. 在「主页设置」中输入 https://starttab.rth1.xyz/
