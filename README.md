# StartTab - 搜索启动页

这是一个简洁美观的搜索启动页，具有以下功能特性：

## 功能特点

- **多搜索引擎支持**：支持必应（默认）、百度、360搜索和纯网址访问
- **设置选项**：
  - 显示/隐藏Logo
  - 显示/隐藏捷径栏
  - 启用/禁用随机壁纸
  - 设置默认搜索引擎
- **随机壁纸**：从img文件夹中随机选择图片作为半透明背景
- **快捷方式管理**：
  - 内置常用网站快捷方式
  - 支持添加自定义快捷方式
  - 自动获取网站图标
  - 右键快捷方式栏可快速添加新快捷方式

## 使用方法

1. 在搜索框输入关键词或网址进行搜索
2. 点击右上角齿轮图标进入设置页面
3. 右键点击快捷方式栏可添加新的快捷方式
4. 点击已有快捷方式可直接访问对应网站

## 技术实现

- 前端：HTML, CSS, JavaScript
- 后端：PHP云函数用于获取图片列表
- 数据存储：使用localStorage保存用户配置和快捷方式

## 文件结构

- `index.html` - 主页面
- `style.css` - 样式文件
- `script.js` - JavaScript逻辑
- `list_images.php` - 获取图片列表的云函数
- `favicon.ico` - 网站图标
- `img/` - 存放背景图片
